qpmr.plot.basic module#

Basic root plots#

qpmr.plot.basic.roots(roots, ax=None, **kwargs)#

Plot roots in the complex plane.

Stable roots (negative real part) are green, unstable red, near imaginary axis blue. Style 'matlab' uses filled circles.

Parameters:

  • roots (NDArray) – Complex roots to plot.

  • ax (Axes) – Matplotlib axes. Created if None.

  • tol (float, optional) – Tolerance for classifying real part as zero (default 1e-10).

  • style (str, optional) – Plot style, 'matlab' or default cross markers.

Returns:

ax – Matplotlib axes with the plot.

Return type:

Axes
