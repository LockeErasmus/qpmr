Notes

  1. vectorized version easily obtainable

qpmr.numerical_methods.mueller_method.mueller(f, x0, x1=None, x2=None, tol=1e-06, max_iter=100)#

Attempt to solve \(f(x) = 0\) via Müller’s method.

Uses a quadratic interpolating polynomial through three successive iterates. Scalar inputs are not yet implemented; array inputs are solved element-wise.

Parameters:

  • f (Callable) – Function to find a root of.

  • x0 (float | NDArray) – First initial guess.

  • x1 (float | NDArray) – Second initial guess. If None, obtained by perturbing x0.

  • x2 (float | NDArray) – Third initial guess. If None, obtained by one secant update.

  • tol (float) – Stopping tolerance. Default is 1e-6.

  • max_iter (int) – Maximum number of iterations. Default is 100.

Return type:

tuple[float | complex | NDArray, bool]

Returns:

  • x (ndarray) – Approximate root(s).

  • converged (bool) – True if the method converged within max_iter iterations.

Raises:

NotImplementedError – If x0 is a scalar.

qpmr.numerical_methods.newton_method.newton(f, f_prime, x0, m=1.0, tol=1e-06, max_iter=100)#

Attempt to solve \(f(x) = 0\) via Newton’s method.

Parameters:

  • f (Callable) – Function to find a root of.

  • f_prime (Callable) – Derivative of f.

  • x0 (float | NDArray) – Initial guess(es) for the root.

  • m (float | NDArray) – Root multiplicity factor for quadratic convergence. Default is 1.0.

  • tol (float) – Stopping tolerance on |f(x)| and step size. Default is 1e-6.

  • max_iter (int) – Maximum number of iterations. Default is 100.

Return type:

tuple[float | complex | NDArray, bool]

Returns:

  • x (float, complex, or ndarray) – Approximate root(s).

  • converged (bool) – True if the method converged within max_iter iterations.

qpmr.numerical_methods.newton_method.numerical_newton(func, x0, tolerance=1e-07, max_iterations=100)#

Vectorized Newton refinement (QPmR v2 compatibility).

Uses central finite differences for the derivative when no analytic f_prime is supplied.

Parameters:

  • func (Callable) – Vectorized complex function.

  • x0 (NDArray) – Initial root guesses.

  • tolerance (float) – Convergence tolerance. Default is 1e-7.

  • max_iterations (int) – Maximum iterations. Default is 100.

Return type:

tuple[NDArray, bool]

Returns:

  • x (ndarray) – Refined roots.

  • converged (bool) – Whether convergence was achieved for all elements.
